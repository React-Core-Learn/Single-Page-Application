# SPA(Single-Page-Application) 구현
## 🚀 학습 목표

- SPA를 이해하고 이점과 사용하는 이유를 설명할 수 있다.
- SPA를 기반으로 동작하는 React의 작동 원리를 깊게 이해할 수 있다.
- CSR(클라이언트 사이드 렌더링)의 특징을 이해하고 활용할 수 있다.
